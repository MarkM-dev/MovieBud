MovieBud is a movie library management application.
Online movie searches are performed via the Rotten Tomatoes api (no longer accepting private api calls).

Can be viewed here (the app is unpublished):
https://play.google.com/store/apps/details?id=markm.moviebudfree


Note: Rotten Tomatoes no longer accepts nor distributes api keys - therefor the app will not function as intended.
